# DiversWebSite — портал учёных

Ruby on Rails приложение для просмотра отчётов об устрицах, собранных через бота DiversBot (MAX). Использует **общую PostgreSQL-базу** `divers_data`.

## Возможности

- **Карта** (OpenLayers) Чёрного и Азовского морей с точками наблюдений
- **Фильтры**: дата, тип встречи, способ указания места, глубина, только избранное
- **Краткая карточка** при клике на точку → переход к полному отчёту с фото
- **Избранное** — добавление отчётов и просмотр на карте / в отдельном разделе
- **Регистрация по токену** — админ создаёт токены, учёные регистрируются
- **Админ-панель** — управление токенами и удаление аккаунтов

## Быстрый старт

```bash
cd DiversWebSite
cp .env.example .env
# заполните DATABASE_URL и ADMIN_PASSWORD

bundle install
bin/rails db:migrate
bin/rails db:seed
bin/rails server
```

Откройте http://localhost:3000 и войдите с `ADMIN_EMAIL` / `ADMIN_PASSWORD`.

## Роли

| Роль | Возможности |
|------|-------------|
| **admin** | Карта, избранное, создание/удаление токенов, удаление пользователей |
| **scientist** | Карта, избранное, просмотр отчётов |

## Регистрация учёного

1. Админ: **Токены** → «Создать токен»
2. Передать токен учёному
3. Учёный: **Зарегистрироваться по токену** → имя, email, пароль

## Фото отчётов

Фото хранятся как токены вложений MAX. Для отображения укажите `MAX_BOT_TOKEN` в `.env` (тот же, что у бота). Если API недоступен, показывается заглушка с подписью.

## Структура таблиц

Приложение **читает** таблицы бота: `reports`, `report_photos`.

**Создаёт** свои таблицы: `scientists`, `registration_tokens`, `favorites`.
